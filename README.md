# Projeto-filmes
Aplicativo web de watchlist, é possível criar uma conta e ter uma lista de filmes ja assistidos, contabilizando a quantidade de horas e minutos gastos.
dependencias usadas: 
- sequelize
- express
- nodemon
- express-handlebars
- body-parser
- express-session
- connect-flash
- bcryptjs
- passport
- passport-local

Api usada:
TMDB - https://www.themoviedb.org/?language=pt-BR
